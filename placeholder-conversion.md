# 标准占位符转换规则

本文档只描述翻译系统中的标准占位符与 Android / iOS 占位符之间的转换关系。

## 标准占位符语法

标准占位符统一使用 `{{` 和 `}}` 包裹。

```text
{{%[参数顺序][宽度][.精度]类型}}
```

| 组成部分 | 示例 | 说明 |
|---|---|---|
| `%` | `%` | 占位符起始符 |
| 参数顺序 | `1`、`2`、`12` | 表示第几个参数 |
| 宽度 | `02`、`03`、`04` | 表示最小显示宽度，常用于补零 |
| 精度 | `.1`、`.2`、`.3` | 表示浮点数保留的小数位数 |
| 类型 | `s`、`d`、`i`、`f`、`b`、`c` | 表示占位符数据类型 |

## 语法判定规则

| 规则 | 示例 | 含义 |
|---|---|---|
| 只有类型 | `{{%s}}`、`{{%d}}` | 普通占位符 |
| 数字不以 `0` 开头 | `{{%1s}}`、`{{%2d}}`、`{{%12f}}` | 数字表示参数顺序 |
| 数字以 `0` 开头 | `{{%02d}}`、`{{%03i}}` | 数字表示宽度或补零格式 |
| 包含 `.数字` | `{{%.1f}}`、`{{%.2f}}` | 数字表示浮点数精度 |
| 包含 `数字$.数字` | `{{%1$.2f}}`、`{{%2$.1f}}` | 同时表示参数顺序和浮点数精度 |
| `%%` | `{{%%}}` | 表示百分号本身 |

## 基础占位符

| 标准占位符 | 含义 | Android | iOS |
|---|---|---|---|
| `{{%s}}` | 字符串 | `%s` | `%@` |
| `{{%d}}` | 整数 | `%d` | `%d` |
| `{{%i}}` | 整数 | `%i` | `%i` |
| `{{%f}}` | 浮点数 | `%f` | `%f` |
| `{{%b}}` | 布尔值 | `%b` | `%b` |
| `{{%c}}` | 字符 | `%c` | `%c` |
| `{{%%}}` | 百分号 | `%%` | `%%` |

## 带顺序的占位符

`%` 后面的数字表示参数顺序，支持任意正整数。

| 标准占位符 | 含义 | Android | iOS |
|---|---|---|---|
| `{{%1s}}` | 第 1 个字符串参数 | `%1$s` | `%1$@` |
| `{{%2s}}` | 第 2 个字符串参数 | `%2$s` | `%2$@` |
| `{{%12s}}` | 第 12 个字符串参数 | `%12$s` | `%12$@` |
| `{{%1d}}` | 第 1 个整数参数 | `%1$d` | `%1$d` |
| `{{%2d}}` | 第 2 个整数参数 | `%2$d` | `%2$d` |
| `{{%1i}}` | 第 1 个整数参数 | `%1$i` | `%1$i` |
| `{{%2i}}` | 第 2 个整数参数 | `%2$i` | `%2$i` |
| `{{%1f}}` | 第 1 个浮点数参数 | `%1$f` | `%1$f` |
| `{{%2f}}` | 第 2 个浮点数参数 | `%2$f` | `%2$f` |
| `{{%1b}}` | 第 1 个布尔参数 | `%1$b` | `%1$b` |
| `{{%2b}}` | 第 2 个布尔参数 | `%2$b` | `%2$b` |
| `{{%1c}}` | 第 1 个字符参数 | `%1$c` | `%1$c` |
| `{{%2c}}` | 第 2 个字符参数 | `%2$c` | `%2$c` |

## 宽度与补零

以 `0` 开头的数字表示宽度或补零格式，不表示参数顺序。

| 标准占位符 | 含义 | Android | iOS |
|---|---|---|---|
| `{{%02d}}` | 整数，最小宽度 2，不足补 0 | `%02d` | `%02d` |
| `{{%03d}}` | 整数，最小宽度 3，不足补 0 | `%03d` | `%03d` |
| `{{%04d}}` | 整数，最小宽度 4，不足补 0 | `%04d` | `%04d` |
| `{{%02i}}` | 整数，最小宽度 2，不足补 0 | `%02i` | `%02i` |
| `{{%03i}}` | 整数，最小宽度 3，不足补 0 | `%03i` | `%03i` |
| `{{%04i}}` | 整数，最小宽度 4，不足补 0 | `%04i` | `%04i` |

## 浮点数精度

`.` 后面的数字表示保留的小数位数。

| 标准占位符 | 含义 | Android | iOS |
|---|---|---|---|
| `{{%.1f}}` | 浮点数，保留 1 位小数 | `%.1f` | `%.1f` |
| `{{%.2f}}` | 浮点数，保留 2 位小数 | `%.2f` | `%.2f` |
| `{{%.3f}}` | 浮点数，保留 3 位小数 | `%.3f` | `%.3f` |

## 带顺序的浮点数精度

| 标准占位符 | 含义 | Android | iOS |
|---|---|---|---|
| `{{%1$.1f}}` | 第 1 个浮点数参数，保留 1 位小数 | `%1$.1f` | `%1$.1f` |
| `{{%1$.2f}}` | 第 1 个浮点数参数，保留 2 位小数 | `%1$.2f` | `%1$.2f` |
| `{{%1$.3f}}` | 第 1 个浮点数参数，保留 3 位小数 | `%1$.3f` | `%1$.3f` |
| `{{%2$.1f}}` | 第 2 个浮点数参数，保留 1 位小数 | `%2$.1f` | `%2$.1f` |
| `{{%2$.2f}}` | 第 2 个浮点数参数，保留 2 位小数 | `%2$.2f` | `%2$.2f` |
| `{{%2$.3f}}` | 第 2 个浮点数参数，保留 3 位小数 | `%2$.3f` | `%2$.3f` |

## 说明

| 规则 | 说明 |
|---|---|
| `{{%2d}}` | 表示第 2 个整数参数 |
| `{{%02d}}` | 表示整数最小宽度为 2，不足时用 `0` 补齐 |
| `{{%2s}}` | Android 转为 `%2$s`，iOS 转为 `%2$@` |
| `{{%2f}}` | Android 和 iOS 都转为 `%2$f` |
| `{{%2$.2f}}` | Android 和 iOS 都转为 `%2$.2f` |
